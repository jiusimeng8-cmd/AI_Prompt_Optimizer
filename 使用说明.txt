AI Prompt Optimizer - 使用说明
================================

## 应用简介
AI Prompt Optimizer 是一个 Windows 桌面工具，用于快速优化文本内容。
支持全局快捷键选中文本后直接优化，结果自动替换到原位置。

## 主要特性
✓ 单文件 exe，无需安装 Python 环境
✓ 支持托盘最小化（关闭窗口不退出）
✓ 全局快捷键（默认 Ctrl+Shift+O）
✓ 自定义 API 配置（兼容 OpenAI 格式）
✓ 多提示词管理
✓ 开机自动启动（可选）

## 文件说明
- AI_Prompt_Optimizer.exe   主程序（约 49MB）
- icon.ico                    应用图标
- 使用说明.txt               本文件

## 首次使用
1. 双击运行 AI_Prompt_Optimizer.exe
2. 点击"设置"按钮，配置 API：
   - Base URL: 你的 API 地址
   - API Key: 你的密钥
   - Model: 模型名称
3. 保存设置后即可使用

## 使用方法

### 方法一：主窗口
1. 在输入框粘贴文本
2. 选择提示词（可自定义）
3. 点击"发送优化"
4. 结果显示后点"复制"

### 方法二：全局快捷键（推荐）
1. 在任意软件中选中文本
2. 按快捷键（默认 Ctrl+Shift+O）
3. 等待 1-3 秒，优化结果自动替换选中文本

## 快捷键修改
1. 点击"设置" → "快捷键"标签
2. 点击"重新捕获快捷键"
3. 按下你想用的组合键
4. 保存

## 提示词管理
1. 点击主窗口的"管理"按钮
2. 左侧列表管理提示词
3. 右侧编辑名称和内容
4. 内容支持变量：
   - [A] = 用户输入
   - [B] = 选中的原始文本
   - [C] = AI 生成的结果

## 开机自启
1. 点击"设置" → "通用"标签
2. 勾选"开机自动启动"
3. 保存

## 托盘功能
- 关闭窗口后程序最小化到托盘（系统右下角）
- 双击托盘图标重新打开窗口
- 右键托盘图标 → 退出 可完全关闭程序

## 注意事项
1. 首次运行可能被 Windows Defender 拦截，选择"仍要运行"即可
2. 全局快捷键需要管理员权限才能在某些系统窗口生效
3. API 配置错误会导致优化失败，请检查网络和密钥
4. 建议将 exe 放在固定位置，避免移动后自启失效
